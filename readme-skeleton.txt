# playing symfony when you having just flask

* `python3-virtualenv` -- project containment
* `flask` -- application structure, routing, cli
* `bootstrap3`, `jquery`, `datatables`, `toastr`
* `flask-debugtoolbar`
* `pytest`, `webtest` -- testing
* `flask_sqlalchemy`
* `flask_wtforms`
* `https://github.com/carlnewton/bootstrap-wtf/` -- bootstrapforms
* `sqlalchemy_json` -- store list without special db support
* `psycopg2-binary`
